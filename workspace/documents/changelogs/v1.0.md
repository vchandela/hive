# Changelog v1.0

## New Features

DataStack v1.0 is the initial release of the analytics platform. Key features include:

- **SQL query engine**: Run SQL-like queries against your data with time-series extensions.
- **Dashboard builder**: Create visual dashboards with line charts, bar charts, and tables.
- **CSV import**: Upload CSV files directly through the web dashboard.
- **API access**: Full REST API with bearer token authentication for programmatic access.
- **Python SDK**: Official Python SDK for easy integration.
